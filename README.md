# FinRelief AI – AI-Powered Debt Relief & Financial Recovery Platform

## Overview

FinRelief AI is a full-stack web application that assists borrowers in managing loans, analyzing financial health, and generating AI-powered negotiation strategies and settlement emails. The platform combines financial analysis with Google Gemini AI to provide intelligent debt settlement assistance through a simple and user-friendly interface.

---

## Features

- User Registration & Login (JWT Authentication)
- Add, View and Delete Loans
- Financial Health Analysis
- Settlement Prediction
- Debt Repayment Timeline
- AI Negotiation Strategy Generation
- AI Negotiation Email Generation
- AI History Management
- Know Your Rights Module
- Profile Management

## Installation

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

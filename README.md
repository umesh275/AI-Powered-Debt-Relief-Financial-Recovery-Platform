# AI-Powered Debt Relief & Financial Recovery Platform

## Overview
This project is an AI-based financial recovery platform that helps borrowers analyze their financial health, predict debt settlements, and generate AI-powered negotiation strategies.

## Technology Stack
- Frontend: React + Vite
- Backend: FastAPI
- Database: SQLite
- AI: Google Gemini API
- ORM: SQLAlchemy

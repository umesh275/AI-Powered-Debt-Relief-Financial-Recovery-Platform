# Project Prerequisites

## Overview

This document outlines the software, tools, frameworks, and services required to develop and run the AI-Powered Debt Relief & Financial Recovery Platform successfully.

## Software Requirements

* Python 3.11+
* Visual Studio Code
* Git
* Google Chrome (or any modern web browser)

## Development Frameworks

* FastAPI
* React.js
* Vite
* SQLAlchemy ORM
* SQLite

## AI Service

* Google Gemini API

## Version Control

* Git
* GitHub

## Authentication

* JWT (JSON Web Token)

## Development Environment

The project requires Python for backend development, FastAPI for REST APIs, React.js with Vite for the frontend, SQLite with SQLAlchemy for database management, Google Gemini API for AI-powered features, GitHub for version control, and Visual Studio Code as the development environment.

## Status

* Python Installed
* Visual Studio Code Installed
* Git Installed
* GitHub Repository Created
* Google AI Studio Account Created
* Gemini API Key Generated
